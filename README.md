## **ZSH Configuration**

Installation:

```sh
cd ~
git clone --recursive git@github.com:degete/zshrc.git ~/.zsh
ln -s .zsh/zshrc .zshrc
ln -s .zsh/zshenv .zshenv
cp .zsh/zshrc.local .zshrc.local
```
